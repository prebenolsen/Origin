## Replace them with general language progression

Continue increasing communicative ability instead of introducing niche vocabulary domains.

Suggested progression:

1. Visiting Spain
2. Meeting People
3. Unlock Spanish/Building sentences
4. Everyday Life
5. Revisiting 'Visiting Spain' <--
Future:. Talking About the Past
Future:. Talking About the Future
Future:. Describing the World
Future:. Talking about home (e.g. Norway, Bergen)

The first modules taught "tourist survival". Now the learner has:

basic verbs
ser/estar
connectors
descriptions
routines
frequency

So the next phase should not just add random vocabulary. It should make old situations conversational.
